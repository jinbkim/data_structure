## 2. 클래스 
### 2.1) C++에서의 구조체
- typedef 선언 없이도 struct를 안쓰고 구조체 선언 가능
- 구조체 안에 함수 삽입 가능
### 2.2) 클래스와 객체 
#### 접근제어 지시자
- public : 어디서든 접근허용
- protected : 상속관계일때, 유도 클래스에서의 접근허용
- private : 클래스 내에서만 접근허용
#### 구조체와 클래스의 차이점
- 구조체는 접근제어 지시자가 없으면 public로 간주
- 클래스는 접근제어 지시자가 없으면 private로 간주
