### 개요 및 동기
- 기본적인 자료 구조에 익숙하다고 가정
- 자료구조의 장단점과 시간 및 공간 복잡도에 대한 이해가 필요
### 내장된 라이브러리가 있는 선형 자료 구조
1) 정적 배열 : 인덱스를 통해 자료에 접근하려 할때 사용
2) 정렬 : merge정렬, heap정렬, quick정렬 
3) 연결리스트 : 원소에 접근하는것이 비효율적
4) 스택 : last in frist out(LIFO)
5) 큐: first in first out(FIFO). 덱 : 양방향 큐
### 내장된 라이브러리가 있는 비선형 자료 구조
6) 이진 탐색 트리: 왼쪽서브트리 노드 키 < 루트 노드 키 < 오른쪽 서브트리 노드 키
7) 힙(우선순위 큐) : 완전이진트리
