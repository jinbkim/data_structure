## 2. 자료구조
### 2.1) 배열
- 연속된 메모리 공간을 차지하는 같은 데이터형들의 집합
#### 함수인자로 배열 넘기는법
- char a[2] -> char a[]
- int b[3][4] -> int b[][4]
- double c[5][6][7] -> double c[][6][7]
### 2.2) 연결리스트
- 실제로 정보를 담고있는 하나의 단위인 노드와 노드의 위치를 저장하는 리스트로 구성
- 배열과 다르게 동적인 자료구조
- 머리와 꼬리를 정의하여 코드의 일관성을 유지 할수있음
#### 단순 연결리스트
- 정보를 저장하는 노드와 다음노드를 가리키는 링크로 구성됨
#### 원형 연결리스트
- 꼬리노드가 머리노드를 가리키는 연결리스트
