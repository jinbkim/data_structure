## 1.개요 및 C언어
### 1.1) 유클리드 알고리즘 (최대 공약수 찾기)
- gcd(u, v) = gcd(u - v, v)
- gcd(u, v) = gcd(u % v, v)
### 1.2) 소수를 구하는 알고리즘
- 2부터 제곱근 까지의 수를 나누어 나머지가 모두 0이 아니면 소수
- 에라토스테네스의 체 : n 이하의 모든 소수 찾기
### 1.3) 포인터
- 메모리는 1바이트 단위로 쪼깨짐
- 포인터는 주소값과, 주소값이 저장된 데이터형으로 구성됨
#### void 포인터
- void 포인터 변수는 *연산을 할수 없음
- 형변환을 하면 *연산 가능
- 데이터형에 관계없이 동작하는 함수를 제작하는데 쓰임
