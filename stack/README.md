#### 스택
- LIFO (Last In, First Out)
- push : 스택의 top에 데이터 추가 
- pop : 스택의 top에 있는 데이터 삭제
- peek : 스택의 top에 있는 데이터 출력
#### 스택 사용법 
- 데이터의 개수가 정해져있고, 데이터의 삽입과 삭제가 많지 않으면  -> 배열기반 스택 
- 데이터의 개수가 정해져 있지않고, 데이터의 삽입과 삭제가 많으면  -> 연결리스트기반 스택
#### 계산기 프로그램 구현을 위해 필요한 알고리즘 
- 전위표기법 : 연산자를 먼저 표시하고, 피연산자를 나중에 표기하는 방법
  ex) ((A+B)*(C-D))(중위표기법) -> *(+(AB)-(CD)) ->*+AB-CD
- 후위표기법 : 피연산자를 먼저 표시하고, 연산자를 나중에 표기하는 방법
  ex) ((A+B)*(C-D))(중위표기법) -> ((AB)+(CD)-)* -> AB+CD-*
- 중위표기법 : 연산자를 두 피연산자 사이에 표기하는 방법
  ex) *+AB-CD(전위표기법) -> *(+(AB)-(CD)) -> (A+B)*(C-D)
      AB+CD-*(후위표기법) -> ((AB)+(CD)-)* -> (A+B)*(C-D)
1. 중위 표기법을 후위표기법으로 변환
2. 피연산자는 스택으로 이동
3. 연산자를 만나면 스택에서 두개의 피연산자를 꺼내어 계산
4. 계산결과는 다시 스택으로 이동
