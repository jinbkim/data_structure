## 공부법

- 어쩌면 사실이 아닐지도 모르지만, 이렇게 믿어야 내가 성실해 질꺼같음.
- 할땐하고 놀땐 놀고가 아니고, 왠만하면 그냥 계속 해.
- 가끔 놀아야 재밌음.
- 집중될때만 하는게 아니고, 집중이 안되도 일단 하다보면 집중이됨.
