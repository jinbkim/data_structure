## 1. 그래프 용어
- 정점(vertex) : 연결의 대상
- 간선(edge) : 정점 사이의 연결선
- 완전 그래프 : 각각의 정점에서 다른 모든 정점을 연결한 그래프
- 가중치 그래프 : 간선에 가중치 정보가 담긴 그래프
- 부분 그래프 : 기존 그래프의 일부	 정점 및 간선으로 이루어진 그래프
- 단순 경로 : 동일한 간선을 중복하지 않는 경로
- 싸이클  : 단순 경로이면서 시작과 끝이 같은 경로
## 2. 인접 리스트 기반 그래프
### (void) graph_init(t_al_graph *graph, int vertex_num)
- 그래프의 정점, 간선 초기화
- 그래프의 정점의 개수만큼 리스트를 만들고 초기화
### (void) add_edge(t_al_graph *graph, int from, int to)
- 두정점에 연결리스트를 이용하여 간선 추가
- 간선 개수 증가
## 3. depth first search(DFS)
### (void) graph_init(t_al_graph *graph, int vertex_num)
- 그래프의 정점, 간선 초기화
- 그래프의 정점의 개수만큼 리스트를 만들고 초기화
- 그래프의 방문기록 배열을 정점의 크기만큼 만들고 모두 0으로 초기화
### (int) visit_vertex(t_al_graph *graph, int vertex)
- 정점의 방문기록이 있으면 0 반환
- 방문기록이 없으면 방문기록을 1로 바꾸고 방문한 정점 출력하고 1 반환
### (voide) dfs_graph(t_al_graph *graph, int vertex)
- 시작 정점을 방문하고 스택에 넣음
- 해당 정점의 리스트를 따라서 방문기록이 없는 정점을 만나면  
  스택에 해당 정점을 넣음
- 직전까지 방문기록이 없었던 정점을 시작으로 리스트를 따라 탐색 반복
- 리스트가 끝에 도달할때까지 방문기록이 없는 정점을 못찾으면
  스택에서 정점을 꺼내어 탐색 반복
- 스택이 비어있다면 모든 방문기록을 0으로 초기화 


### 신장 트리
- 그래프의 모든 정점이 간선에 의해서 하나로 연결되어 있음 
- 그래프 내에서 사이클을 형성하지 않음
- 최소 신장 트리 : 신장트리의 모든 간선의 가중치 합이 최소인 그래프
### 크루스칼 알고리즘
- 가중치를 기준으로 간선을 내림차순으로 정렬
- 높은 가중치의 간선부터 하나씩 그래프에서 제거 
- 두 정점을 연결하는 다른 경로가 없을경우, 간선 제거 x 
- 간선의 수 + 1 = 정점의수 이면 최소 신장 트리 완성
